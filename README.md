# Cine_pluss
Aplicación básica de cobro para un cine creada en python con wxglade. Consta de 4 ventanas: inicio de sesión en el que colocas tu nombre de usuario y tu clave para poder entrar y pasar a la siguiente ventana, Taquilla en ella eliges la pelicula, la hora en la que deseas verla cuantos boletos de adultos y menores, tambien la dulceria y el ticket.
